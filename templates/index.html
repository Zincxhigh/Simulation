Getting Started With Emulation

Learn how to play classic games on modern devices

What is Emulation?

Emulation allows your computer to imitate older gaming systems such as NES, Game Boy, PlayStation, or Nintendo DS. This lets you play classic games without needing the original console.

ROMs

Game files used in emulators are called ROMs or ISOs. These are digital copies of cartridges or discs.

  • NES / SNES → .nes / .sfc
  • Game Boy → .gb / .gba
  • PlayStation → .iso / .bin
  • Nintendo DS → .nds

Most people create ROMs from their own games using dumping tools.

Emulators

Different consoles require different emulator software.

  • GameCube → Dolphin
  • Nintendo Switch → Ryujinx
  • PlayStation 2 → PCSX2
  • PlayStation 3 → RPCS3

After installing the emulator, open it and select the ROM file to start playing.
