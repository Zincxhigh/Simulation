PSX 80 Minute Patcher

Patch PS1/PS2 Games for 700MB CD-R Compatibility

About

PSX80MP is a tool by Alex Free that patches PlayStation game images to work on early PS2 models when burned to 80 minute / 700MB CD-Rs.

It fixes a hardware bug found in early PS2 consoles that causes crashes or boot failures when using higher capacity discs.

Download

  • psx80mp-v2.0.1-windows-x86_64-static.zip Portable Release For x86_64 Windows (64 bit)

  • psx80mp-v2.0.1-linux-i386-static.zip Portable Release For Linux 3.2.0 and above, 386 CPU minimum (32 bit)

  • psx80mp-v2.0.1-linux-i386-static.deb Deb package file For Linux 3.2.0 and above, 386 CPU minimum (32 bit)

  • psx80mp-v2.0.1-linux-x86_64-static.zip Portable Release For x86_64 Linux 3.2.0 and above (64 bit)

  • psx80mp-v2.0.1-linux-x86_64-static.deb Deb package file for x86_64 Linux 3.2.0 and above (64 bit)

How It Works

Early PS2 consoles expect lower capacity discs. When using 700MB discs, the laser can overshoot data, causing read errors.

PSX80MP fixes this by adding dummy data (CDDA silence sectors), preventing the laser from reading invalid areas.

Usage

Drag & Drop Method

Drag your BIN file onto the executable.

Command Line

psx80mp <data_track.bin>

Note: Usually patch track 1. If it fails, try track 2.

Previously, users had to rely on smaller CD-Rs (650MB). This tool allows modern 700MB discs to work reliably on affected systems.

PS3 HFW + HEN Guide

Install Hybrid Firmware & Enable Homebrew

  • Go to Settings → System Settings → System Information
  • If version ≤ 4.78 → update required
  • If 4.80–4.82 → continue
  • If 4.83–4.92 → install HFW
  • If > 4.92 → not supported

Step 2: Install HFW

  • Download latest HFW
  • Verify MD5 hash
  • Install like normal firmware update
  • DEX users → install Dual Boot HFW

Step 3: Prepare Browser

  • Open Internet Browser
  • Triangle → Tools
  • Clear cookies, cache, history
  • Set homepage to blank

Install PS3HEN

Method 1 (ps3xploit)
  • Open browser → go to ps3xploit.me
  • Add bookmark
  • Select PS3HEN → HEN Auto Installer
  • Wait for download & auto install
  • Restart console
  • Launch ★ Enable HEN
Method 2 (ps3addict)
  • Go to ps3addict.github.io
  • Select Alternate HEN Installer
  • Follow on-screen instructions
  • Use Remote Play trick
  • Install HEN
Method 3 (USB)
  • Download PS3HEN.BIN
  • Place on FAT32 USB
  • Insert into right USB port
  • Open browser → visit installer link
  • Install and reboot

After Installation

  • Use ★ Enable HEN every boot
  • Package Manager will appear
  • Install homebrew apps
  • Works offline after setup

Troubleshooting

Error 80710A06

Fix date & time via internet

Error 80029564

Retry installation

Crashes or fails?

Retry multiple times or reinstall HFW

Final Notes

PS3HEN allows homebrew similar to CFW, but some features like CCAPI are not supported. You must enable HEN every time you boot your PS3.

PS5 Jailbreak Guide

Current Status & Exploits (FW ≤ 5.50)

Overview

PS5 jailbreak is currently available for firmware 5.50 and below using kernel exploits.

Lower firmware versions (especially ≤ 4.51) have the best support including homebrew and backups.

Supported Firmware

  • 1.00 → 4.51 Best Support
  • 5.10 → 5.50 Partial support
  • Above 5.50 Not supported

Kernel Exploits

UMTX Exploit (1.00 → 5.50)
  • Based on FreeBSD vulnerability (2024)
  • Supports firmware 1.00 → 5.50
  • Full jailbreak via etaHEN on ≤ 4.51
  • 5.00+ still improving
IPV6 Exploit (3.00 → 4.51)
  • Released in 2022
  • Gives full kernel access
  • Works best on 4.03 / 4.50 / 4.51

Other Exploits (Not Full Jailbreak)

BD-JB Exploit (≤ 7.61)

Allows limited native code execution via Blu-ray vulnerability.

Mast1c0re Exploit (≤ 7.60)

Runs PS2 ISOs and homebrew via PS2 emulator.

Entry Points

  • WebKit exploit (browser-based)
  • Blu-ray exploit (disc-based)
  • ESP8266 host method

⚠️ Scam Warning

If a website claims a jailbreak for the latest PS5 firmware — it is almost always fake.

  • Real exploits are publicly known in the scene
  • Fake sites use ads or malware
  • If it sounds too good → it’s a scam

Known Vulnerabilities Timeline

  • 2024 – UMTX Kernel Vulnerability
  • 2023 – Hypervisor exploit
  • 2022 – Mast1c0re exploit
  • 2022 – Kernel + WebKit exploit
  • 2022 – IPv6 exploit
  • 2021 – WebKit font exploit
  • 2020 – SMAP bypass

Buying a Hackable PS5

Look for consoles with firmware ≤ 4.51 for best compatibility.

Avoid updating firmware after purchase.

Unlock Homebrew & Custom Firmware

Overview

All PS Vita models can be hacked using HENkaku. This allows homebrew, plugins, and custom firmware features.

By default, the jailbreak is not permanent, but installing Ensō makes it persistent.

Check Firmware

  • 3.60 / 3.65 (Chinese text) → Already modded
  • 3.65 / 3.68 / 3.74 → Continue with HENlo

HENlo Jailbreak

Steps

  • Enable JavaScript
  • Clear cookies & history
  • Press Unlock My Vita
  • Install:
    • HENkaku
    • VitaDeploy
  • Exit to LiveArea

Special Cases

PCH-1xxx without Memory Card

Use "Replace NEAR with VitaDeploy" option.

South Korean Models

Use SD2Vita as internal storage.

Post Setup

Internal Memory Setup

Install Ensō (Permanent Hack)

Steps
  • Open VitaDeploy

PSP ARK-4 CFW Guide

Install Modern Custom Firmware

Overview

ARK-4 is the most modern custom firmware for PSP, offering enhanced compatibility, features, and performance.

It can be installed as temporary or permanent (recommended).

Requirements

  • Firmware 6.60 or 6.61
  • Memory Stick or internal storage
  • USB connection to PC

Install Temporary CFW

Steps
  • Download ARK-4
  • Connect PSP via USB
  • Extract:
    • ARK_01234 → PSP/SAVEDATA
    • ARK_Loader → PSP/GAME
  • Launch ARK Loader from Game menu
  • PSP will restart with CFW enabled

Note: Temporary CFW must be launched again after reboot.

ARK-4 Features

  • Supports PRO Online & XLink Kai
  • NoDRM engine for ISO games
  • PS1 game support with configs
  • Region-free UMD playback
  • Custom launcher included
  • Supports legacy homebrew
  • cIPL support on all PSP models

Install Permanent CFW (cIPL)

Steps
  • Copy ARK_cIPL to PSP
  • Launch cIPL installer
  • Select New cIPL
  • Done — CFW loads on boot

This method enables automatic CFW at startup.

Full Flash Installation

Steps
  • Copy ARK_Full_Installer
  • Run installer
  • Follow on-screen instructions

This installs ARK-4 directly to internal memory, removing reliance on save files.

Other CFW Options

PRO CFW

Popular, supports online play and plugins.

(L)ME CFW

Faster install, supports legacy homebrew via LEDA.

Important Notes

  • Temporary CFW resets after reboot
  • cIPL = best permanent option
  • Full flash removes dependency on save files
  • PSP Go has internal storage advantage
